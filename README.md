# Celsius2026GoodReadsCompare
Comparador de autores que van al Celsius232 contra tu biblioteca de Goodreads

Básicamente exportas el CSV de tu biblioteca de Goodreads y la subes a la pagina.
La pagina compara los autores de tu biblioteca de Goodreads con los asistentes al evento y te devuelve las coincidencias.
